# Automated Surveillance System using Arduino Nano + ESP32-CAM

IoT-based intrusion detection system with real-time email alerts and image capture.

## Tech Stack
- **Microcontrollers**: Arduino Nano (ATmega328), ESP32-CAM
- **Sensors**: HC-SR04 Ultrasonic, SG90 Servo Motor
- **Protocols**: UART, SMTP over TLS
- **Languages**: C/C++ (Arduino IDE)

- Real-time image capture on intrusion
- Email notifications with image attachments
- Visual (laser) and audio (buzzer) alerts
